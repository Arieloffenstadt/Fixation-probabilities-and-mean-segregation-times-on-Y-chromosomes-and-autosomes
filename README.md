# Fixation-probabilities-and-mean-segregation-times-on-Y-chromosomes-and-autosomes
Auxiliary files to [Article Link]


The notebook `Diffusion_approx.ipynb` computes the fixation probabilities and mean segregation times derived from the diffusion approximations of the article, and the script `ScriptMutationFixationTimeRate_Auto.slim` performs the corresponding empirical computations using individual-based SLiM simulations.
